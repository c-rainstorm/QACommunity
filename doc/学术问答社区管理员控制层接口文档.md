## 登录

### 检测管理员登录信息

- url: /checkAdminLogin.action
- parameters:
  - id; 
  - password
- return:
  - 成功后重定向至网站主页
  - 失败后重定向登录页面
- side-effect:
  - 登录成功后，设置session属性
    - admin
      - adminLoginStatus = true
      - id = $admin.id$
      - name = $admin.name$
  - 登录失败后，设置session属性
    - admin
      - adminLoginStatus: false

- url: /shutdownQuestion.action
- parameters:
  - question_id
- return:

- option:
